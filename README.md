# Web Development Project

## Overview
This project is a 3-page website created using HTML and CSS for the Unit 3 Web Development Project.

## Pages
- Home (index.html)
- Places page
- Contact/Experience page

## Features
- Navigation bar linking all pages
- Styled using CSS
- Images stored in the images folder
- Clean and structured layout

## How to Open the Website
1. Download the repository
2. Open the folder
3. Double-click index.html to open the website in a browser
